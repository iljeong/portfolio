중고차 플랫폼 Gcar 앱 리뷰 데이터 37,923건을 수집·분석하여 사용자 이탈 원인을 밝히는 데이터 분석 프로젝트입니다. 낮은 앱스토어 평점과 지속적인 사용자 불만이 있었지만, 어떤 기능이 문제인지 정확히 파악되지 않은 상태였습니다.

KoELECTRA 기반 감정 분류 모델을 적용하고, SHAP으로 주요 불만 키워드를 추출했습니다. 고객 여정을 단계별로 매핑하여 이탈이 집중되는 지점을 시각화하고 서비스 개선안을 제안했습니다.

분석 결과 사용자 불만의 68%가 '검색·필터 기능'과 '가격 신뢰도' 두 영역에 집중됨을 확인하였으며, 우선순위 개선 로드맵을 도출했습니다.
