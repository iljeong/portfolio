강사 분석 → 커리큘럼 설계 → 페이지 IA → 런칭 매니징

신규 강의 6개 핵심 성과

문제

새로운 강사 영입 시 콘텐츠 방향, 타겟, 커리큘럼 구성, 페이지 기획이 담당자 경험에만 의존하고 있었다.

역할

콘텐츠 상품 기획자

결과

신규 강의 6개 기획·출시 완료 (7개월, 2024.05–12)

01 · Problem

새로운 강사 영입 시 콘텐츠 방향, 타겟, 커리큘럼 구성, 페이지 기획이 담당자 경험에만 의존하고 있었다. 제작 프로세스가 문서화되지 않아 일정 지연과 품질 편차가 반복됐다.

02 · Approach

  1. 01

    강사·타겟 분석

    강사 콘텐츠 히스토리·SNS 분석 + 경쟁 강의 비교를 통한 차별화 포지션 도출

  2. 02

    커리큘럼 설계

    학습 목표 정의 → 차시 구조화 → 난이도 흐름 설계 (강사 협업)

  3. 03

    페이지 IA 설계

    기존 구매 페이지 클릭 흐름 분석 → 경쟁사 벤치마킹 → 콘텐츠 이해도 우선 IA 재설계

  4. 04

    제작 협업

    촬영·디자인·편집팀과 마일스톤 기반 일정 관리, 품질 체크리스트 운영

  5. 05

    프로세스 문서화

    전 과정을 Notion 기반 표준 템플릿으로 정리, 후임자 인수인계 완료

03 · Results

런칭

신규 강의 6개 기획·출시 완료 (7개월, 2024.05–12)

페이지 개선

상세페이지 구성 방식 개선, 수강생 이해도 기준 내부 피드백 반영

프로세스

표준 제작 프로세스 도입, 담당자 간 품질 편차 감소 및 온보딩 시간 단축

04 · Contribution

Role
