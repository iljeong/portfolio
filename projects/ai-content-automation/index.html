AI 자동화 · 콘텐츠 기획 · 2026.03 — 2026.05

AI 파이프라인으로 콘텐츠 제작 시간을 83% 단축했다

Claude AI + Figma MCP 연동 → 29개 피드 자동 생성

Claude AIFigma MCPPythonGoogle Sheets자동화콘텐츠 기획
3시간 → 30분 핵심 성과

문제

패션 브랜드 인스타그램 피드를 자료조사 → 전략 수립 → 카피 작성 → 디자인 적용 순서로 수작업 시 브랜드당 평균 3시간 이상 소요됐다.

역할

콘텐츠 기획자 · 파이프라인 구조 설계자 (AI 실행 도구 활용, 1인 프로젝트)

결과

브랜드당 평균 제작 시간 3시간 → 30분 이내, 약 83% 단축 (29개 브랜드 평균 기준)

01 · Problem

패션 브랜드 인스타그램 피드를 자료조사 → 전략 수립 → 카피 작성 → 디자인 적용 순서로 수작업 시 브랜드당 평균 3시간 이상 소요됐다. 품질 일관성도 담당자에 따라 편차가 컸다.

02 · Approach

  1. 01

    파이프라인 설계

    자료조사 → 전략 → 카피 생성 → 디자인 자동 적용 4단계 구성. 각 단계 입출력 스펙 정의

  2. 02

    AI 연동

    브랜드 분석 기준·카피 포맷 설계 후 Claude AI로 실행, Figma MCP로 디자인 자동 반영 (수동 Figma 작업 제거)

  3. 03

    품질 게이트

    fact_check·QA 체크리스트 자동화로 게시 전 오류 검증 (인물 권리·브랜드 정보 정확도 포함)

  4. 04

    대시보드 구축

    Google Sheets로 39개 피드 진행 현황·캡션·발행 상태 통합 관리

  5. 05

    SEO 전략

    캡션 v4 포맷 + 3-Tier 해시태그 전략 설계·적용 (발행 피드 전수 적용 완료)

03 · Key Insight

01

병목은 이미지 선정이 아니라 브랜드별 규칙 재설계의 반복이었다

매 피드마다 브랜드 톤·캡션 포맷·이미지 크롭 기준을 처음부터 결정하는 것이 진짜 시간 낭비였다. 기준을 한 번 설계하면 AI가 반복할 수 있다.

02

AI 실행 전에 인간의 설계 레이어가 있어야 품질이 안정된다

Claude API에 그냥 위임하면 결과가 들쑥날쑥하다. 브랜드 분석 기준·캡션 구조·출력 포맷을 먼저 설계한 뒤 AI를 실행했을 때 일관성이 생겼다.

03

자동화의 진짜 효과는 속도보다 담당자 의존도 제거다

3시간 → 30분 단축보다 중요한 건, 누가 하더라도 같은 품질이 나온다는 것. 이 프로젝트의 목표는 속도가 아니라 재현 가능한 프로세스 설계였다.
